# homepage
weiguang homepage

# NodeJS 部署方法
1.安装NodeJS
2.npm install -g express  安装express
3.npm install -g express-generator 4.0后需要安装这个才能CMD中使用express
4.express -t ejs homepage 生成项目信息
5.npm install 安装package.json下的资源


# NodeJS使用
1.在项目目录下运行npm install 自动安装package.json下的资源
